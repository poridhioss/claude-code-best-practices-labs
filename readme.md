# Claude Code: From Setup to Orchestration

A hands-on course for learning Claude Code — Anthropic's CLI-based coding assistant.

You'll start by understanding how coding assistants work under the hood, then progressively build real configurations — CLAUDE.md files, custom commands, hooks, skills, and agents — on top of a starter project.

## Starter Project

Students clone a simple Node.js weather API and build Claude Code configurations on top of it throughout the course.

```bash
git clone https://github.com/poridhioss/claude-code-best-practices-starter.git
```

## Modules

| Module | Title | Topics |
|--------|-------|--------|
| [Lab 1](lab_1/readme.md) | Introduction | Coding assistants, tool use, agentic loop, parallelization, model tiers, plan mode |
| [Lab 2](lab_2/readme.md) | Getting Started | Installation, CLAUDE.md, context management, custom commands, MCP servers, GitHub |
| [Lab 3](lab_3/readme.md) | Hooks & SDK | Hook events, defining and implementing hooks, decision control, headless mode, CI/CD |
| [Lab 4](lab_4/readme.md) | Agents & Skills | Skills, agent isolation, frontmatter configuration, orchestration patterns |
